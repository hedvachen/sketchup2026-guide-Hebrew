📋 Copy — העתקה
איך משתמשים
  1. בחרו את האובייקט עם Select
  2. לחצו M ואז Ctrl (הסמן מציג +)
  3. בחרו נקודת ציר
  4. בחרו יעד
  5. הקלידו מרחק ב-VCB → Enter
טיפים מתקדמים
3x → 3 עותקים נוספים
/3 → חלוקה שווה
